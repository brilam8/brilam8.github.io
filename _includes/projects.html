My projects

These are some of the passion projects that I have taken on throughout my programming career. Some of these projects are polished and complete while others remained prototypes that I could learn and build off of in the future. I primarily focus on back-end development, integrating databases, and unit testing built APIs. Hover over each project to find out more!

PING!

An network analyzer that checks your connection to the internet.

PING!

Developed using Android Studio and Java, published on the Google Play Store. Now has over 600+ active users.

Analyzes one's network connection to various servers or IP addresses and analyzes the gathered data, presenting it to the user in a more organized format to allow for easy interpretation of one's connection.

GreustBot

A multipurpose Discord bot that encourages interactions between server members on Discord.

GreustBot

Built with JavaScript integrating libraries Node.js and Discord.js. Designed to run on a Raspberry Pi that utilizes PM2 to manage and keep the Discord bot online 24/7.

Contains over 30 commands that encourages user interaction, retrieves and displays data from various APIs, and handles a virtual currency system within Discord servers and among its members.

ACM Splash Pages

A web application that generates splash pages at ease for events hosted by ACM UCSD.

ACM Splash Pages

Handled the back-end codebase with Express, MongoDB, and unit testing with Jest.

Documented the back-end API, MongoDB models, and endpoints at which you can send POST, GET, DELETE, and MODIFY requests.

Engaged in bi-weekly Scrum sprints with team members to develop a web application that would auto-generate splash pages for any ACM hosted event.

Showcased an initial prototype to the ACM UCSD Board with 30+ attendees.
